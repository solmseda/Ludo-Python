Para rodar os testes deve-se abrir e arquivo Testes.py e rodar de acordo com a ide utilizada:
Exemplo para a produção dos testes foi utilizado o VSCode, nesta ide deve-se pressionar Ctrl + F5 para rodar o programa.
OBS: Os testes estão rodando através de Mockup pois um bug impediu de testas as funções do modulo main.py, quando se tentava testar ele abria a interface gráfica e não realizava os testes

Para testar a interface gráfica, rode o módulo main.py, uma janela irá se abrir:
1 - Ao clicar em jogar
	1.1 - na tela que surgir digite a quantidade de jogadores desejada (minimo 2 e máximo 4)
	1.2 - digite o nome do primeiro jogador e clique no botao ao lado para escolher a cor (cada clique mostra uma das 4 cores disponiveis)
	1.3 - repita o passo (1.2) para os jogadores seguintes
	1.4 - caso clique em jogar (nestes momento irá fechar o programa pois o jogo ainda não foi implementado)
	1.5 - caso clique em voltar ao menu irá retornar a tela anterior
2 - Ao clicar em regras
	1.1 - uma tela mostrando as regras do jogo irá aparecer
	1.2 - para fechar essa tela clique no X no canto superior da mesma
3 - Histórico ainda não implementado
4 - Ao clicar em sair o programa encerrara.
